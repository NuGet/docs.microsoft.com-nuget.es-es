<span data-ttu-id="953fb-124">En Visual Studio, puede establecer estos valores en las propiedades del proyecto (haga clic con el botón derecho en el proyecto en el Explorador de soluciones, elija **Propiedades** y seleccione la pestaña **Paquete**).</span><span class="sxs-lookup"><span data-stu-id="953fb-124">In Visual Studio, you can set these values in the project properties (right-click the project in Solution Explorer, choose **Properties**, and select the **Package** tab).</span></span> <span data-ttu-id="953fb-125">También puede establecer estas propiedades directamente en los archivos del proyecto (`.csproj`).</span><span class="sxs-lookup"><span data-stu-id="953fb-125">You can also set these properties directly in the project files (`.csproj`).</span></span>
