## <a name="search-syntax"></a><span data-ttu-id="e6135-198">Sintaxis de búsqueda</span><span class="sxs-lookup"><span data-stu-id="e6135-198">Search Syntax</span></span>

<span data-ttu-id="e6135-199">La búsqueda de paquetes NuGet funciona igual en nuget.org, desde la CLI de NuGet, y desde la extensión Administrador de paquetes NuGet en Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="e6135-199">NuGet package search works the same on nuget.org, from the NuGet CLI, and within the NuGet Package Manager extension in Visual Studio.</span></span> <span data-ttu-id="e6135-200">En general, la búsqueda se aplica a palabras clave, así como a descripciones de paquetes.</span><span class="sxs-lookup"><span data-stu-id="e6135-200">In general, search is applied to keywords as well as package descriptions.</span></span>
